ScholarHub Africa Site Plan

Site Name

ScholarHub Africa

This website helps African students find scholarships, internships, and remote jobs.

Site Purpose

The website provides scholarship information, internship opportunities, remote jobs, and educational resources for students.

Scenarios

  • What scholarships are available for African students?
  • How can I find remote jobs related to my field?

Color Scheme

  • Navy Blue (#003366) – Headings and footer.
  • Light Gray (#F4F4F4) – Background color.

Typography

  • Poppins – Headings.
  • Roboto – Body text.
